Detailed Description

Namespace for modifier constants.

For each modifier constant there is a jpcre2::Uint option value. Some modifiers may have multiple values set together (ORed in bitwise operation) and thus they may include other modifiers. Such an example is the 'n' modifier. It is combined together with 'u'.
